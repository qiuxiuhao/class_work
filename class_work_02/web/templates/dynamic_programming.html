📊 动态规划算法

北邮学生选课优化问题

算法说明

问题描述

学生需要在有限的时间内选择课程，使得总学分最大化，同时避免时间冲突。这是一个带时间约束的0-1背包问题。

算法设计

  • 状态定义：dp[i][mask] 表示考虑前i门课程，占用时间段为mask时的最大学分
  • 状态转移：对于第i门课程，选或不选（检查时间冲突）
  • 时间复杂度：O(n × 2^m)，n为课程数，m为时间段数
  • 空间复杂度：O(n × 2^m)

参数设置

建议：5-20门（数量越多，计算时间越长）
设为0表示不限制
